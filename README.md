项目名称：hahamusic
=====
配置
==

服务器：wampserver(php-7.3.1 apache-2.4.37 mysql-5.7.24)
编辑器：vscode
后端框架：thinkphp5.1
前端框架:vue+element-ui
数据库：mysql

运行
===
client路径下，输入npm run dev
