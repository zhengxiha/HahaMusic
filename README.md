项目名称：hahamusic
=====
配置
==

服务器：wampserver(php-7.3.1 apache-2.4.37 mysql-5.7.24)<br>
编辑器：vscode<br>
后端框架：thinkphp5.1<br>
前端框架:vue+element-ui<br>
数据库：mysql<br>

运行
===
  client路径下，输入npm run dev
